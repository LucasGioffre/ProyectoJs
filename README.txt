E-Commerce de Instrumentos Musicales

Este proyecto trata de una simulacion completa de un Carrito de Compras de Instrumentos Musicales.

Funcionalidades:

- Visualización de productos en cards con imagen, descripción y precio.
- Carrito de compras donde se pueden agregar y eliminar productos.
- Uso de libreria Toastify para indicar que se agregan productos al carrito.
- Cálculo automático del total de la compra.
- Uso de JSON para almacenar y gestionar los productos.
- Formulario que se debe completar para poder finalizar la compra correctamente.
- Botón "Finalizar Compra" que muestra el total de la compra.

Tecnologias Utilizadas:

- HTML: Estructura del contenido de la tienda.
- CSS: Diseño y estilos para hacer el sitio visualmente atractivo.
- JavaScript: Lógica para el carrito de compras y otras interacciones.
- JSON: Para manejar los datos de los productos.
